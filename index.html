Homeworks :

10.1. User card 10.2. Getting even numbers from an array 10.3. Contact book

10.1. User card

Create an object that contains information about the user, such as name, age, location, etc. Create an object method to retrieve and display this data.

10.2. Getting even numbers from an array

You have an array of numbers. Use the existing array methods to create a new array that contains only the even numbers from the original array.

10.3. Contact book

Create an object that has the same property as the array of objects. These represent the contacts in your contact book. Each object should contain a name, phone number, and email address. Add a method to search for a contact by name and a method to add new contacts.
